### python环境搭建
#### pygame 安装
 #### 1. windows
 1.  在cmd后中输入下面代码可以获取到pip支持的文件名还有版本
 ```
import pip; print(pip.pep425tags.get_supported())
 ```
 2. 根据支持版本下载[pygame download](https://www.lfd.uci.edu/~gohlke/pythonlibs/#pygame)
 3. 下载好之后将文件放到项目目录下安装执行
 ```
 pip install filename
 ```
