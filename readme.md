#### python依赖库版本确定(以pygame为例)
 1.  在cmd后中输入下面代码可以获取到pip支持的文件名还有版本
 ```
import pip; print(pip.pep425tags.get_supported())
 ```
 2. 根据支持版本[pygame url](https://www.lfd.uci.edu/~gohlke/pythonlibs/#pygame)
 3. 复制文件地址fileurl,并执行
 ```
	pip install --user fileurl
 ```
